1.Calculator :-Write a program to make a simple calculator using switch statement that takes operator and two numbers from user and print the result after operation.

Answer:-

        const Calculator = (a, b, c) => 
{

  switch(a)
  {
    case "+":
    return b+c;
    case "-":
    return b-c;
    case "*":
    return b*c;
    case "/":
    return b/c;
  }
};
